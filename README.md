# Maytalz

[Maytalz](https://maytalz.netlify.app) is a private playground with [Nuxt](https://nuxt.com/) and [Contentful](https://www.contentful.com/).

# Tools

- Framework: Nuxt
- Headless CMS: Contentful
- Styling: Tailwind CSS
- Hosting: Netlify
